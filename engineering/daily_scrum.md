## 每日代碼上傳 Git Commit

所有軟體工程師每日至少Git Commit一次到自己的local branch，讓主管：

* 查看代碼進度
* 對錯誤方向調整方向，讓錯誤方向最多持續不超過一天
* 確保代碼都有持續備份到雲端Git Repo
